# business-card    ![GithubActions](https://github.com/molvqingtai/business-card /workflows/build/badge.svg)


电子名片

> 技术栈：Vue + Koa + MongoDB



### Install

**client**

```
cd client && npm install
```
**server**

```
cd server && npm install
```



### Run

**client**

``` shell
cd client && npm run serve
```

**server**

```shell
cd server && npm start
```
